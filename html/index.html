HTML and CSS

With these two technologies, you can create structured pages with beautiful styles!

HTML5

HTML 5 is a revision of the Hypertext Markup Language (HTML), the standard markup language for describing the contents and appearance of Web pages. HTML5 was developed to solve compatibility problems that affect the current standard, HTML4.

CSS3

Cascading Style Sheets Level 3 (CSS3) is the iteration of the CSS standard used in the styling and formatting of Web pages. CSS3 incorporates the CSS2 standard with some changes and improvements. A key change is the division of standard into separate modules, which makes it easier to learn and understand.
